# 欢迎！

这是一个基于 **Vite + Vue 3** 的极简博客框架。

## 怎么写新文章

1. 在 `src/posts/` 目录下新建一个 `.md` 文件
2. 顶部写 frontmatter（标题、日期、摘要）：

```md
---
title: 文章标题
date: 2026-07-16
excerpt: 一句话摘要
---

正文用 Markdown 写……
```

3. 推送到 GitHub，Actions 会自动重新部署。

> 就这么简单，开始写吧！
